## About

MindMania is an online platform for building and taking timed quizzes and exams. It supports question creation, user management, and result tracking — suitable for educators, trainers, or anyone who needs to run assessments online.

---

## Features

- Create and manage quizzes/exams
- Timed tests and automatic submission
- Multiple question types (MCQ, short answer — extendable)
- User authentication and role-based access (students/administrators)
- Score calculation and reporting
- Responsive UI (mobile & desktop)
- Deployment-ready (live demo hosted)
